Implementation of the algorithm described in:
Y. Koren, L. Carmel and D. Harel,
ACE: A Fast Multiscale Eigenvector Computation for Drawing
Huge Graphs

This package contains the files:
ACE_paper.pdf             
    --   the paper describing the algorithm
ACE.exe              
    --   executable (for running on a Windows PC)
params.txt   
    --   parameters file (command line argument to ACE.exe)
ChacoFileFormat.pdf    
    --   description of Chaco's graph file format
         http://www.cs.sandia.gov/~bahendr/chaco.html
         
showgraph.m
    -- A Matlab utility for displaying graphs

\Workspace             
    --   a Visual C++ workspace with all the sources of the algorithm
\Graphs
    -- Several sample graphs, from public web resources 
       
To execute the program write:
   ACE  paras.txt
